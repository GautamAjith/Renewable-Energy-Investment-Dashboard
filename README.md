# Renewable-Energy-Investment-Dashboard

I built this dashboard to explore which countries may have strong potential for renewable energy investment. The project uses data from the World Bank and brings together indicators such as renewable energy consumption, electricity access, CO₂ emissions per capita, and GDP per capita.

The idea behind the dashboard is simple: countries with rising energy needs, lower electricity access, higher emissions, or growing economic capacity may be important places to study for future renewable energy opportunities. Instead of looking at these indicators separately, the dashboard combines them in one place so users can compare countries more easily.

The dashboard includes an interactive world map, country trend charts, regional filters, CO₂ comparisons, and an investment opportunity view. Users can select different years, regions, and countries to understand how renewable energy patterns have changed over time.

Through this project, I practiced working with real public data, cleaning and reshaping country-level datasets, merging multiple indicators, and building an interactive dashboard using Python, Pandas, Plotly, and Streamlit.

Overall, this project connects my interest in economics, sustainability, and data analysis by showing how publicly available data can be used to better understand global renewable energy trends and investment possibilities.
